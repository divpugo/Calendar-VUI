HELP AREA

You can communicate with the system by voice or chat

1. For voice interaction. click the microphone icon found at the bottom right

2. You can also write commands in the chat section which pops up

Accepted Commands

⚫ To display this help pannel: "help, guide, assist, commands"

⚫ To navigate to UniHub: "home. homepage, main"

⚫ To navigate to MDXplay and access recorded lectures: "play, lecture, lectures, video, videos, recording, recordings"

⚫ To display next month on calendar: "next month, month after"

⚫ To display previous month on calendar: "previous month, month before"

⚫ To navigate to today's date on calendar: "today, now"

⚫ To prompt system to add a reminder: "reminder"

⚫ To prompt system to save reminder: "save, add, submit"

⚫ To prompt system to explain how to add reminder: "how, create"

⚫ To prompt system to explain how to delete a reminder: "delete, erase"

⚫ To close pop-ups: "close"
